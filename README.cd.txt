# Voice Assistant Project

This is a simple Python Voice Assistant program that listens to voice commands and performs tasks.

## Features
- Speech recognition
- Open websites (Google, YouTube)
- Search information from Wikipedia
- Tell the current time

## Technologies Used
- Python
- SpeechRecognition
- pyttsx3
- Wikipedia library

## Installation

Install required libraries:

pip install SpeechRecognition
pip install pyttsx3
pip install wikipedia
pip install pyaudio

## Run the Program

python voice-assistant.py
